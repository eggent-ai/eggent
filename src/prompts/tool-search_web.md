# Web Search Tool

Search the internet for current information.

## When to Use

- Need up-to-date information (news, releases, current events)
- Technical documentation or API references
- Facts you're not confident about
- Information that changes frequently
- Research tasks

## Best Practices

- Use specific, targeted search queries
- For technical queries, include technology names and versions
- Review multiple results before drawing conclusions
- Cite sources when presenting information from search results
- If initial results aren't helpful, try rephrasing your query
